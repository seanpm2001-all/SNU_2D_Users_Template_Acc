SNU developer log day 379

I procrastinated and did other stuff for most of today, and began to update the site tonight. After a few minutes of CSS research, I found some properties to start using. I also had some things I wanted to do today.

I livened up SNU with a CSS feature I haven't discovered yet, and I am happy with it. I will try to learn more CSS for the rest of my 5 day weekend (4 more days) and forward. Today, a lot was done, and I worked for over 2 hours. The update log is getting massive as well, I haven't noticed any severe performance issues while editing, and it is very close to 2.10 Megabytes in size (even with UNIX (LF))

Today was a pretty good day for development.

SNU developer log day 382

I had more time today, but I didn't spend too much time with site development. I did some experiments with CSS buttons today, just trying to figure out more advanced CSS. The test I made isn't entirely functional yet.

Today I worked on CSS testing, a new Sourcebase update, the usual documentation, and an update to the SNU Dialup center. I plan on doing a small update tomorrow to the SNU repositories, bringing in an older project, as I need to take it easy, since school starts back up again on Tuesday.
