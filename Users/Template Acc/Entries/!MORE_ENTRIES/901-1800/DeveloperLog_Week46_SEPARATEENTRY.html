SNU developer log day 319

I had a small development day today, I was frequently out of focus. At least I had some development time this weekend.

With the way the site is currently written, I feel like it is going to be a huge piece of laughing stock for r/programmerhumor for a while. I am still just applying the base, I am not doing the best job at it though.

Overtime, the site is becoming more advanced, new improvements are made on a monthly basis, and I have been completely self-taught on all of this.

Today I added in 0 dimensional moder, and 4D mode. Both modes are easter eggs, 4D mode will be more useful than 0D mode. 0D mode is a joke, as you can't view 0 dimensional content. 4 dimensional mode is going to be an upgrade on 3D mode for those who love tesseracts.

I have stated on the 4D mode page that there likely won't be a 5D mode unless the idea is supported, but the option is definitally not off the table. I just don't have a use in it yet. If anyone really wants it, I am willing to go up to 9D mode max, after that I will likely stop accepting new mode ideas.

Why 9D? I like the 2 byte directory names, and it can be 0D to 9D. I still won't go above 4D unless it gets support.

Today was a decent day with development, but not entirely the best.

Sean Patrick Myrick on December 15th 2019

SNU developer log day 322

I created a big documentation update today, improving the page source code level, regular documentation, and extensive increases to the HTTP list pages. I also ported over a CSS experiment with text shadows.

Thanks to my extensive research lately, I have found and added many new tags and elements to pages to make the site a lot more modern. The process for compatibility modes (especially with Internet Explorer) will downgrade features for those browsers.

It is crazy how a browser that was once dominating the world at the time fell so short on hundreds of HTML, CSS, and JS elements for so long, with some prominent features still not supported to this day with either IE or Edge.

Later in the evening, I exported 5 scripts. I was going to do this another day, but I didn't want to hold it off any further.
